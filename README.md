# 🏥 Hospital Management System

A Full Stack Hospital Management System built using ASP.NET Core MVC, Web API, SQL Server, and AI Chatbot integration.

This system helps manage hospital operations including patients, doctors, appointments, billing, and reports with secure authentication.

---

## 🚀 Features

- Patient Management
- Doctor Management
- Appointment Booking
- Department Management
- Billing System
- Reports Dashboard
- Role Based Authentication
- AI Chatbot (Google Gemini)
- Dark Mode UI

---

## 🧱 Tech Stack

### Frontend
- ASP.NET Core MVC
- Bootstrap
- HTML / CSS / JavaScript

### Backend
- ASP.NET Core Web API
- Entity Framework Core

### Database
- SQL Server

### AI Integration
- Google Gemini API
